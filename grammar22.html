Lesson 2 Grammar 2

Before, we have learned the basic sentence structure: Subject object verb (SOV).

We also learned topic marker ( 은 and 는) which is commonly attached to the subject.

In Korean, objects also have markers, 을/를, which is attached at the end of the object.

Here is how to use them in a sentence:
[Topic + Topic Marker] then [Object + Object Marker] then [verb].

If the object ends in a vowel, attach -를.
If the object ends in a consonant, attach -을.

Examples (don’t worry about the verbs we haven’t learned yet):
*notice the topic particles in orange!

  • 나는 밥을 먹습니다
  • I eat rice (object)
  • 이 사람은 그림을봅니다
  • This man looks at the picture (object)
  • 줄리아는 우유를마십니다
  • Julia drinks milk (object)
